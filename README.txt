PROJECT NAME:
House Rent & Utility Management System

DESCRIPTION:
This is a web-based system developed to manage house renters.
Renters can login to view their rent, electricity bills, payment status,
and payment history. Admin can manage renters, add bills, mark payments,
and view monthly reports.

TECHNOLOGIES USED:
Frontend: HTML, CSS, Bootstrap
Backend: PHP
Database: MySQL
Server: XAMPP (Apache + MySQL)

MODULES:
1. Renter Login
2. Renter Dashboard
3. Admin Login
4. Admin Dashboard
5. Rent Management
6. Electricity Bill Management
7. Payment Tracking
8. Monthly Report

HOW TO RUN PROJECT:
1. Install XAMPP
2. Start Apache and MySQL
3. Place project folder in htdocs
4. Import database using phpMyAdmin
5. Open browser and go to:
   http://localhost/renter-system/login.php
